# Dejyra Website

**迪极拉（Dejyra）官网项目**  
**Software systems for the future.**

这是 Dejyra / 迪极拉 的官方网站项目，采用静态页面方式构建，适用于 GitHub Pages 直接部署。

项目整体风格以极简、沉浸式、未来科技感为核心，适合用于品牌展示、技术组织宣传、项目导航与联系方式展示。

---

## 项目简介

Dejyra 是一个面向未来的软件技术品牌，关注以下方向：

- 软件系统
- AI 工具
- 开发者平台
- 自动化工程
- 数据与信息处理
- 技术品牌展示

本项目用于承载 Dejyra 的公开官网内容，包括品牌首页、About 页面、Projects 页面与 Contact 页面。

---

## 页面结构

当前站点包含以下页面：

- `index.html`：首页
- `about.html`：关于 Dejyra
- `projects.html`：项目方向与仓库展示
- `contact.html`：联系页面
- `404.html`：错误页

---

## 资源文件

站点资源位于 `assets/` 目录，主要包括：

- Logo
- Favicon
- Apple Touch Icon
- Open Graph 分享图
- Web Manifest
- 微信二维码图片
- 站点样式资源
