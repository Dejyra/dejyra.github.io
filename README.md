# Dejyra Website Project

这是可直接部署到 GitHub Pages 的完整官网项目，已包含：

- 首页 `index.html`
- 关于页 `about.html`
- 项目页 `projects.html`
- 联系页 `contact.html`
- logo / favicon / og-image / manifest
- 微信二维码图片
- `robots.txt`
- `sitemap.xml`
- `404.html`

## 直接使用方式

1. 解压项目。
2. 将全部文件上传到 `dejyra.github.io` 仓库根目录。
3. 打开仓库 `Settings -> Pages`
4. 选择：
   - Branch: `main`
   - Folder: `/(root)`

## 需要你替换的内容

- `hello@dejyra.com` 替换成你的真实邮箱
- 如果 GitHub 组织名变化，替换 `https://github.com/Dejyra`
- 根据真实项目调整 `projects.html` 中的仓库名称与描述

## 资产目录

所有图标和图片都在 `assets/` 目录中。
